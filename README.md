ember-cli-spinkit
==============================================================================

an ember-cli addon for [SpinKit](https://tobiasahlin.com/spinkit/)


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install @onehilltech/ember-cli-spinkit
```
